IPF-Editor

Getting Started

  Use this is the simple scenario  to browse Image Producers and Images from existing Image Producer Factory

      Load  Image Producer Factory (IPF).   Select   File| Open| Image_Producer_Factory from the main menu.  Choose IPF. (IPF is actually a directory). For example, choose EXAMPLES\IP_EXAMPLES_FROM_BOOK\FUNCTIONAL_LANGUAGE\IPF_COLOR2. New IPF object will appear in IPF Pane in the left part of the main window

      Open Image Window.  File| Open| Open_Image_Window   from main menu. New Image Window will appear in Image Window Pane

      Select node “a” and press Build button on IPF Window. New Image Producer will appear in Image Producer Pane.

       Press Paint button on the Image Window. You will get something like this:

 

Figure 1.1

 

      You  can create many Image Producers and use them for painting complex pictures. Paint button of the Image Window creates new image using currently selected Image Producer.

      Press left mouse button  and drag mouse on the Image Window to set new painting coordinate system.

      Use Mutate and Mutate(Large) buttons on Image Producer Window to mutate the Image Producer and then paint again.

 

Main Window

Figure 1.1

The Main Window includes 4 Panes

 

      Image Producer Factory Pane is used for Image Producer Factory Windows. Image Producer Factory is used to build Image Producers.

      Image Producer Pane is uses for Image Producer Windows. Image Producer is used to paint Image in Image Window.

      Image Window  Pane is used for Image Windows

      Pallet Pane is used for Pallet Windows.

 

2.4      Image Window

Figure 1.5

Image Window is used to display the images, produced by the Image Producer. To create Imag Window use   File|Open|Open_Image_Window . 

      Press left mouse button and drag mouse on the Image Window to set new painting coordinate system.

      To paint the image press “Paint” button.

      To stop painting press “Stop” button.

      To clear window press “Clear” button.

      To choose image background press “Bcgr” button, then press “Clear” button

      To undo last painting press “Undo” button.

      To scale the image press “+” or  “-“  buttons.

 

 

2.2      Image Producer Factory

Image Producer Factory actually is represented as a folder: each file is a node object. Image

It may be one level folder in the case of constrain free IPF   or two level folder. In second case the name of directory is considered as a label.

 

Producer Factory is Loaded (or Created) using File|Open|Image_Producer_Factory

      Node  name can use   “( subfolder1,subfolder2,…)”  description for possible links for children nodes.

      Build button is used to build Image Producer.

      Use right mouse button to copy or delete selected nodes

 

To set maximum value of the Image Producer Tree use Tools|Options and set new value to the “Maximum Depth of Image Producer”

 

2.3      Image Producer

Figure 1.4

 Image Producer is created

      Using File|New|Image_Producer_Node from the list of existing object nodes

      Using Build button of the Image Producer Factory

      Using right mouse button + copy from the Image Producer Factory

 

Image Producer Window has following functions

      Use Mutate button to make geometry mutation of the Image Producer

      Use Mutate(Large) button to make more significant geometry mutation.

      Use Mutate Color button to make color mutation using Active Pallet

      Use Save button to save Image Producer in the current Image Producer Factory.( The Name of the Image producer can be changed in the name text field)

      Use Decomposition button for inductive inference of the new Image Producer Factory

      Use View IP Tree to see internal structure of the Image Producer.

2.5      Pallet  Window

Figure 1.6

 

Image producer uses Pallet window for Color Mutations. During Color Mutation Image Producer randomly choose colors from Pallet and use them for color mutations. To create new pallet use File|New|Pallet. You can add or delete colors from the pallet and save/restore the pallet

1.2      Load Existing Image Producer Factory

To load  Image Producer Factory (IPF)  select   File| Open| Image_Producer_Factory from the main menu.  Choose IPF. For example, choose EXAMPLES\EXAMPLES_IPF\IPF_SimpleTree. New IPF object will appear in IPF Pane in the left part of the main window.

Select node “Start” and press “Build” button on IPF Window.

New Image Producer will appear. Each press to “Build” button will create new “Image Producer”

1.2      Mutations of the Image Producer

Press Mutate button on Image Producer Object to Mutate Image Producer

Press Mutate Large  button on Image Producer Object to perform Large or Structural mutations of the Image Producer

Press Paint button on Image Window to see new Image

 

1.3      Color Mutations the Image Producer

Open Existing Pallet (File| Open | Open Pallet)or Create New Pallet (File| New | New Pallet). Pallet object appears in the Pallet pane.

You may change colors on pallet object, using Delete and Add Color buttons of  the pallet Window

Press Color Mutate button on Image Producer Object to make color mutation of the Image Producer. Colors are selected from the currently selected pallet

 

1.4      Build New Image Producer Factory From Node Objects

 

Use File|New|Image_Producer_Node . Select Nodes from the list.

New Image Producers will appear in Image Producers Pane.(Each Image Producer Contain only one node)

Use select   File| Open| Image_Producer_Factory  to create and open  new empty directory for Image Producer Factory.

Copy each Image Producer to the Image Producer Factory  

 

 

1.4      Build New Image Producer Factory using Inductive Inference

Use select   File| Open| Image_Producer_Factory  to create and open  new empty directory for Image Producer Factory.

Use “decomposition” button of the Image Producer window to build new  Image Producer Factory.

Use Tools|Options to set value of the the Destruction Probability for Image Producer Factory inductive inference (default value 0.5, 0.0 – destroy all links, 1.0 keep all links from Image Producer)
